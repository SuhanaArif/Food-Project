# Food-Project

A simple food delivery landing page made with HTML and CSS.

## What this project contains

- `index.html` - the main page for the website.
- `resources/css/style.css` - custom styles for the page.
- `resources/css/queries.css` - responsive styles for different screen sizes.
- `vendors/css/normalize.css` and `vendors/css/grid.css` - helper styles for layout and browser consistency.
- `resources/img/` - images used on the page.

## What this page shows

- A hero section with buttons to order and learn more.
- A feature section with delivery benefits.
- A meals gallery with food photos.
- A "How it works" section with steps to use the service.
- A cities section showing where the service is available.

## How to use

1. Open `index.html` in a browser.
2. No build tools or server are needed.

## Notes

- This is a static website, so it does not include a backend or database.
- You can edit the text, images, and CSS to change the look and content.
